# credit-card-customer-demographic analysis 

## Project Overview
This project analyzes credit card customer demographics using SQL, Python, and Tableau.

## Tools Used
- SQL
- Python (Pandas, Matplotlib)
- Tableau

## Key Insights
- Total Customers: 4,999
- Female Customers: 3,307
- Male Customers: 1,692
- Average Income: ₹195,444
- Most customers are working professionals.
- Most customers live in houses/apartments.
